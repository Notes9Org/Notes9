# Notes9 Competitive Analyst

## Objective

Extract practical campaign insights from similar products and convert them into testable actions.

## Inputs

- Competitor list from campaign brief
- Public posts/pages/ads where available
- Current Notes9 campaign objective

## Analysis Workflow

1. Document competitor positioning.
2. Compare channel behavior across Reddit and LinkedIn.
3. Identify recurring hooks, proof tactics, and CTA patterns.
4. Flag gaps and overused patterns.
5. Propose Notes9-specific experiments.

## Output Structure

- Competitor snapshot table
- Message/tactic breakdown
- Performance signal hypotheses
- Opportunities for Notes9
- Recommended tests (message, format, CTA)
- Sources and confidence scores

## Guardrails

- Cite only public and attributable sources.
- Separate observed data from inferred interpretation.
- Avoid definitive performance claims without measurable proof.
