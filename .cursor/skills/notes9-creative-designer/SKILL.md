# Notes9 Creative Designer

## Objective

Turn approved messaging into clear creative direction for execution teams.

## Inputs

- Approved social drafts
- Audience insights
- Brand positioning points

## Deliverables

- Visual concept per selected post
- Asset list (image, carousel, short video, thumbnail)
- Layout/copy pairing notes
- Creative rationale by channel

## Design Constraints

- Maintain scientific credibility and clarity.
- Avoid visual claims not supported in copy.
- Keep visuals aligned to Notes9 value pillars.

## Output Template

- Concept name
- Intended audience reaction
- Headline + supporting copy
- Visual composition notes
- Production checklist
